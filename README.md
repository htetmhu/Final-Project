Introduction:
This project implements Student Process Tracker using the Matplotlib library for data visualization and Tkinter for user interface. 

The main functionality of the app includes:
- Nested Data Management: to store multiple students' grades and informations using dictionaries 
- Visual Reporting: generating the charts to visualize the student's process
- Calculations: calculating averages for each student 

Installation:
To run the app locally:
1. Make sure Python 3.x is installed.
2. Install required dependencies:
   pip install matplotlib

How to Run the App:
1. Open terminal/command prompt in the project folder.
2. Run the main script:
   python main.py
3. Controls:
   - Adding new students 
   - Calculating averages
   - data visualization with matplotlib
   - searching/sorting algorithms
